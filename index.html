Sobre o Projeto Ultra Voice.

Projeto que possibilita realizar o preenchimento dos dados de um formulário através do reconhecimento de voz, em conjunto com do sintetizador de fala para orientar no preenchimento destes campos.

O projeto segue em Beta, sendo testado e aprimorado constantemente, logo possíveis bugs podem ocorrer. Se encontrar algo de errado, não hesite em comunicar.
